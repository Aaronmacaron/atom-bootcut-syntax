## Installation

### CLI installation

`apm install bootcut-syntax`

### Install inside Atom

1. Settings > Install
2. Choose "Themes"
3. Click "Search Themes"
4. Type `bootcut-syntax`
5. Hit <kbd>Enter<kbd> (or <kbd>Return</kbd>)
6. Click "Install"
